# Estructura de Imágenes para YaDay Nail Designer

Este directorio contiene las imágenes que se utilizarán en la landing page.

## Estructura de Carpetas

```
public/images/
├── services/       # Imágenes de servicios
├── about/          # Imagen de la diseñadora
└── hero/           # Imagen de fondo del hero (opcional)
```

## Recomendaciones de Tamaño

- **Imágenes de servicios (main):** 800x1000px (proporción 4:5)
- **Imágenes de servicios (muestras):** 600x600px (cuadradas)
- **Imagen del diseñador:** 800x1000px (proporción 4:5)
- **Imagen hero (opcional):** 1920x1080px o mayor

## Formato
- Usa formato JPG o PNG
- Optimiza las imágenes para web (compresión de calidad ~80-85%)
- Asegúrate de que las imágenes estén bien iluminadas y sean de alta calidad

## Nota
Actualmente, la web muestra placeholders donde irán las imágenes. Una vez que agregues tus imágenes a estas carpetas con los nombres especificados, se mostrarán automáticamente en la página.
